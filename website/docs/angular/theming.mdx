# Theming

The appearance of the KritzelEditor can be customized via CSS variables. You can override these variables in your application's stylesheet to match your design system.

## How to use

Typically, you would define the overrides in your global stylesheet within the `:root` selector.

```css
:root {
  --kritzel-primary-color: #007bff; /* Your brand's primary color */
  --kritzel-engine-background-color: #f8f9fa; /* A light grey background */
  /* ... and so on */
}
```

## Available CSS Variables

Below is a list of all available CSS variables that you can use to style the KritzelEditor.

### General

| Variable                  | Default   | Description                                    |
| ------------------------- | --------- | ---------------------------------------------- |
| `--kritzel-primary-color` | `#dd0031` | The primary color used throughout the editor. |

### Engine

This variable controls the background of the drawing canvas.

| Variable                          | Default   | Description                               |
| --------------------------------- | --------- | ----------------------------------------- |
| `--kritzel-engine-background-color` | `#ffffff` | Background color of the editor canvas.    |

### Selection

These variables control the appearance of the selection box and the handles for selected annotations.

| Variable                                   | Default                        | Description                                        |
| ------------------------------------------ | ------------------------------ | -------------------------------------------------- |
| `--kritzel-selection-box-background-color` | `rgba(221, 0, 48, 0.2)`        | Background color of the selection box.             |
| `--kritzel-selection-box-border-color`     | `rgba(221, 0, 48, 0.5)`        | Border color of the selection box.                 |
| `--kritzel-selection-border-color`         | `var(--kritzel-primary-color)` | Border color of a selected annotation.             |
| `--kritzel-selection-border-width`         | `2px`                          | Border width of a selected annotation.             |
| `--kritzel-selection-border-style`         | `solid`                        | Border style of a selected annotation.             |
| `--kritzel-selection-handle-size`          | `6px`                          | Size of the selection handles.                     |
| `--kritzel-selection-handle-color`         | `#000000`                      | Color of the selection handles.                    |
